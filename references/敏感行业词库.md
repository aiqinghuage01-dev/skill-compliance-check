# 敏感行业特殊违禁词库

> 判断是不是敏感行业（大健康/医疗/美业/教育/金融投资），如果是，在通用审核之上再加一轮行业特殊审核。

---

## 敏感行业识别规则

| 学员说的行业关键词 | 归属敏感行业 | 需要查的特殊词库 |
|-----------------|------------|-------------|
| 抗衰、减肥、养生、保健、中医、按摩、康养、医药 | 大健康/医疗 | §1 |
| 美容、美体、美肤、护肤、SPA、美甲 | 美业 | §2 |
| 培训、教育、考证、升学、留学、补习 | 教育 | §3 |
| 金融、理财、财税、投资、股票、基金、知识付费 | 金融/知识付费 | §4 |
| 减肥、塑形、医美、整形、玻尿酸 | 医美 | §1 + §2 |

**判断方法**：从系统记忆读用户行业，如果命中上面的关键词 → 加查行业特殊库。

---

## §1 大健康/医疗/养生

### 1.1 医疗功效词（严禁）

普通食品、保健食品、化妆品都**严禁**使用以下词：

- 治愈、根治、治疗、疗效、药效、对症、主治、适应症
- 抗癌、防癌、抗肿瘤、抗病毒、抗感染
- 消炎、抗炎、抑菌、杀菌、灭菌、抗菌、除菌、防菌
- 清热解毒、清热、解毒、活血、化瘀、排毒、通便、润肠
- 壮阳、补肾、滋阴、养血、补气、养精、固精
- 镇痛、止痛、止血、止咳、化痰、平喘
- 降血压、降血糖、降血脂、降胆固醇、降三高
- 增高、生发、防脱、乌发、黑发

**合规替换**：
- "治愈" → "辅助改善""日常呵护"
- "消炎" → "温和护理"
- "排毒" → "日常代谢""清爽"
- "降血糖" → "关注血糖健康"
- "抗癌" → 不要提，直接删

### 1.2 医学术语（严禁）

**严禁**普通食品/化妆品使用：
- 细胞、激素、骨骼、血液、器官、脏器、肝、肾、心、胃
- 代谢、免疫、再生、修复、激活、焕活
- 神经、细胞级、分子级、DNA、基因
- 内分泌、平衡、调节

**合规替换**：
- "细胞修复" → "肌肤焕亮"
- "激活细胞" → "感受轻盈"
- "免疫增强" → "日常健康"

### 1.3 虚假功效词（严禁）

- 永久、绝对、100%、立竿见影、药到病除、一次见效
- 彻底、完全、根本性解决、终结XX问题
- 医学级、专业治疗级
- 瞬间、即刻（慎用）

### 1.4 禁止暗示疗效的场景

- 前后对比图（"治疗前 vs 治疗后"会被判定为疗效宣传）
- "3天见效""7天瘦5斤"这种具体时间+具体效果承诺
- 患者证言（"吃了这个，我的XX好了"）
- 医生/医院背书（除非真有授权）

## §2 美业/化妆品

### 2.1 化妆品虚假宣传词（严禁）

**严禁**普通化妆品宣传：
- 特效、速效、高效、全效、长效、速白、瞬白、急效
- X天见效、XX见效、XX天有效
- 细胞级、分子级、DNA、基因、干细胞
- 抗衰老（可换成"抗皱""紧致"）、抗氧化
- 活血、化瘀、平衡、调节、代谢
- 纤体、瘦身、减肥、溶脂、瘦腿（普通化妆品不能说）
- 丰胸、丰臀、增高（普通化妆品不能说）
- 美白、祛斑、防晒、祛痘、抗敏（只有特殊用途化妆品可宣称，需要"特字号"批文）
- 生发、防脱、育发、乌发（只有特殊用途化妆品能说）

### 2.2 美业服务话术陷阱

- "办卡送XX"（需要明码标价，不能虚假折扣）
- "终身服务"（无法保证的承诺）
- "专家亲自操作"（需有证件证明）
- "有效率90%"（无数据来源的效率数字）

**合规替换**：
- "美白" → "提亮肤色"（普通化妆品）
- "瘦脸" → "轮廓管理"
- "祛斑" → "淡化肌肤暗沉"
- "立马见效" → "日常使用感受"

## §3 教育/培训

### 3.1 承诺类违规（严禁）

**严禁**：
- 保证通过、100%通过、包过、包录取、包拿证
- 提分保证、XX天提XX分、一次通过
- XX名师授课、升学率100%、清北率XX%
- 保过班、不过退款（需合规合同，不能随意承诺）
- XXX人成功案例（编造数字违规）

**合规替换**：
- "包过" → "高通过率"（需有真实数据）
- "100%通过" → "高达XX%通过率（截至2026年X月数据）"
- "名师" → "资深教师""从业X年教师"

### 3.2 虚假宣传（严禁）

- 虚构专家/名师背景
- 抄袭同行话术
- 伪造学员证言
- 虚构升学案例
- "别的机构都不行"类型的贬低

### 3.3 焦虑营销（限流）

- 不输在起跑线、XX岁是最后机会、黄金期错过就没了
- 不报就废了、不学就落后
- XX孩子都在学，你家孩子不学就…

**合规替换**：
- 改成"建议""可以考虑"等软性表达
- 用"机会"代替"最后机会"

## §4 金融/投资/知识付费

### 4.1 收益承诺（严禁）

- 保证收益、稳赚不赔、零风险、刚性兑付
- 月入XX万、年化XX%、一夜暴富
- 包赚、稳赚、必赚、一定赚钱
- 跟我一起财富自由

### 4.2 投资建议（严禁普通人做）

- 具体推荐XX股票、XX基金
- 明确的买卖时机判断
- 虚构投资案例

### 4.3 知识付费虚假宣传（严禁）

- 学完月入10万、学完年入百万（极端收入承诺）
- 终身受用、永远不过时
- XXXX学员成功案例（虚构数字）
- 只要你学就能成功

**合规替换**：
- "月入10万" → 删除数字，讲方法论
- "跟我学" → "分享经验""交流心得"
- "稳赚" → "合规合法""长期视角"

---

## §5 整形/医美（超高危）

### 5.1 医美机构专用（非医美机构严禁使用）

即使有资质，也需严格：
- 抽脂、隆胸、双眼皮、开眼角、隆鼻、玻尿酸、肉毒素
- 瘦脸针、水光针、埋线
- 祛疤、祛斑（非医疗器械/化妆品不能说）

### 5.2 非医美机构绝对不能说

- 打针、注射、埋线、整形、微整
- 任何医疗操作名词
- 任何手术前后对比
- 任何"治疗"字眼
