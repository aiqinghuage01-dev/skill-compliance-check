# 通用违禁词库（所有行业必查）

> 来源：2026 抖音禁用词大全 + 广告法 + 平台规则
> 任何行业的文案都要过这一轮

---

## 一、极限词（广告法禁用，100%违规）

**严禁使用**：

- **数量级**：国家级、世界级、最高级、第一、唯一、首选、顶级、国家级产品、独家、首家、最新、最先进、第一品牌、金牌、名牌、优秀、顶级、独家
- **销量/地位**：全网销量第一、全球首发、全国首家、全网首发、世界领先、顶级工艺、王牌、销量冠军、第一(No.1/Top1)、极致、永久、掌门人、领袖品牌、独一无二、绝无仅有、史无前例、万能
- **程度**：最高、最低、最、最具、最便宜、最新、最先进、最大程度、最新技术、最先进科学、最佳、最大、最好、最小、货真价实、最新技术、最先进加工工艺
- **权威**：最赚钱、最受欢迎、最新一代、大牌、精确、超薄、领导品牌、领先上市、著名、巨星、奢侈、世界、全国X大品牌之一
- **绝对化**：100%、国际品质、高档、正品、国家级、世界级、最高级、最佳

**合规替换**：
- "第一" → "其中之一""比较好的""用户认可的"
- "最好" → "很好""广受好评"
- "独一无二" → "特色鲜明""与众不同"
- "顶级" → "优质""高品质"

---

## 二、虚假宣传词（严重违规）

**严禁**：
- 包治百病、治愈、根治、神效、奇效、特效、疗效显著
- 100%有效、保证、承诺、绝对、一定、必然
- 纯天然、无毒无害、无副作用、0添加（除非有检测报告）
- 国家XXX领导人推荐、专供、特供、XX大会指定

**合规替换**：
- "100%" → "高达"（去掉绝对化）
- "保证" → "力求""致力于"
- "治愈" → "辅助改善""日常呵护"
- "专供特供" → "定制版""专业款"

---

## 三、刺激消费词（疑似欺骗）

**严禁**：
- 秒杀、抢爆、再不抢就没了、不会再便宜了、万人疯抢、抢疯了
- 恭喜获奖、全民免单、点击有惊喜、点击获取、免费领取、领取奖品
- 非转基因更安全、进口更安全等绝对化表述
- 限时（无具体时限）、随时结束、仅此一次、随时涨价、马上涨价、最后一波

**合规替换**：
- "秒杀" → "限时优惠"
- "抢" → "购""选"
- "免费领取" → "赠送""体验装"
- "随时结束" → 给出**具体截止日期**

## 四、不文明用语（会被限流）

**严禁**：
装逼、草泥马、特么的、操、牛逼、MB、尼玛、卧槽、靠、傻逼、SB、智障、屌、艹、老子、他妈的、我艹、王八蛋、滚、傻叉、狗日、瞎几把、脑残、丢你老母等

**合规替换**：
- 全部删除或替换为中性表达
- "我靠" → "我天"
- "傻逼" → "离谱""不靠谱"

## 五、歧视用语（严重违规）

**严禁**：

## 七、时限用词（必须具体）

**严禁**使用：
随时结束、仅此一次、随时涨价、马上涨价、最后一波、快结束了

**必须**：
- 写具体日期："截止到2026年4月30日"
- 或具体时限："活动持续3天"

## 八、钱财词（高风险）

**提到这些词要特别注意**：
比特币、欧元、美元、泰铢、人民币、虚拟货币、以太币、平民币、英镑、金融、刷单、买卖

**建议**：
- 不讨论具体货币投资
- 不涉及虚拟货币、区块链炒作
- 不提刷单、代刷
